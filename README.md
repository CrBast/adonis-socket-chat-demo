## Basic chat with [AdonisJS websocket](https://adonisjs.com/docs/4.1/websocket)


## Setup

1. Clone repository
2. Install AdonisJS
```bash
> yarn global add @adonisjs/cli
```
4. Configure your `.env`
3. Install dependencies
```bash
> yarn
```
3. Run server
```bash
> adonis serve --dev
```
